# AutoSuggest
A JavaScript plugin to suggest and autocomplete predefined words while typing in input fields.

# Installation
```
npm i @avcs/autosuggest --save
```

```
<script type="text/javascript" src="@avcs/autosuggest/dist/AutoSuggest.js"></script>
```
